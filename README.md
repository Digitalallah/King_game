# King_game

Telegram Mini App адаптер для оригинальной игры «Кинг».

## Важно про оригинал

Графику и правила нельзя воссоздавать приблизительно: чтобы они совпадали с оригиналом на 100%, Mini App не переписывает игру, а запускает оригинальную сборку без изменений внутри Telegram-обертки.

В текущем репозитории оригинальных файлов игры нет. Поэтому нужно добавить их в папку `original/` так, чтобы входная точка была доступна по пути:

```text
original/index.html
```

Все относительные ассеты оригинальной игры должны лежать рядом с ним в той же структуре, в которой они были в исходной игре.

## Что делает адаптер

- Загружает `original/index.html` в полноэкранный `iframe`.
- Не меняет DOM, CSS, графику, ассеты и игровую логику оригинала.
- Инициализирует Telegram WebApp API: `ready`, `expand`, отключение вертикальных свайпов при поддержке клиента, цвета фона и заголовка из темы Telegram.
- Учитывает safe-area отступы мобильных устройств.
- Показывает понятное сообщение, если оригинальная сборка еще не добавлена.

## Локальный запуск

```bash
python3 -m http.server 8080
```

Откройте `http://localhost:8080`.

## Деплой для Telegram Mini App

1. Добавьте оригинальную сборку игры в `original/`.
2. Разместите весь репозиторий на HTTPS-хостинге.
3. В BotFather создайте Mini App и укажите URL на `index.html`.
4. Добавьте ссылку на Mini App в меню бота или кнопку Web App.
